## 🎨 Essentials

[**Figma**](https://www.figma.com/)\
Primary tool for UI design, prototyping, and design systems

```
brew install --cask figma
```

[**FigJam**](https://www.figma.com/figjam/)\
Whiteboarding, workshops, and user flows inside Figma\
&#xNAN;_(included with Figma — no separate install)_

***

### **⚡ Prototyping**

[**ProtoPie**](https://www.protopie.io/)\
Advanced interaction prototyping beyond Figma’s capabilities

```
brew install --cask protopie
```

***

### **🧠 Design Systems**

[**Figma**](https://www.figma.com/)\
Source of truth for components, variants, and design tokens

[**Storybook**](https://storybook.js.org/)\
Developer-facing component documentation and UI testing

[**zeroheight**](https://zeroheight.com/)\
Design system documentation and publishing

***

### 🔬 Research & Testing

[**Maze**](https://maze.co/)\
Rapid testing for prototypes and design validation

[**Lyssna**](https://www.lyssna.com/)\
Lightweight usability testing and design validation

[User Interviews](https://www.userinterviews.com/)\
Participant recruitment for user research

[**Dovetail**](https://dovetail.com/)\
Research repository and insight synthesis

***

### **🗂 Portfolio (Optional)**

[**Framer**](https://www.framer.com/)\
Modern no-code portfolio and marketing site builder

```
brew install --cask framer
```

[**Webflow**](https://webflow.com/)\
Advanced visual web design and CMS platform
