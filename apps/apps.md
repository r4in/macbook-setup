# Intro

Oh, hello there! Here are some of the apps and tools we use daily. Apps we love are marked with a [🏆](https://emojipedia.org/trophy/) emoji in their category. You don’t need to install everything. Take what you need and experiment with what works for you.

## Installation

As much as possible, I install everything using Homebrew because updating and managing apps is faster in Terminal.

```
brew install --cask <app-name>
```

However, not everything is available as a cask (e.g., Adobe Creative Cloud), so there will be a few exceptions. This is why some apps are installed via **mas** (Mac App Store) on the command line.

## Pro Tip

I usually turn off automatic updates by default. That prevents updates from popping up mid-work. At the end of the week, I run OS and app updates. With Homebrew, upgrading everything is as easy as typing this command:

```
brew cu --all
```

Then sit back, relax, and enjoy the show 😎
