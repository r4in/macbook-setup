# Productivity Tools

### 🧠 Core Tools

**ChatGPT**\
AI for writing, structuring, and execution

```
brew install --cask chatgpt
```

**Claude**\
AI for deep thinking, long documents, and nuanced reasoning

```
brew install --cask claude
```

**Raycast**\
App launcher and command center for your entire system

```
brew install --cask raycast
```

**Fantastical 2**\
Powerful calendar with natural language input and multi-calendar support

```
brew install --cask fantastical
```
