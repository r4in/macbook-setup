# Tech Stack

## Ruby

macOS comes with Ruby installed, but you don’t want to mess with core OS files. Use `rbenv` and `ruby-build` to manage and install Ruby versions for your development environment.

## Python

macOS, like Linux, ships with [Python](http://python.org/) already installed. But you don’t want to mess with the system Python (some system tools rely on it, etc.), so we’ll install our own version(s). There are two ways to install Python: (1) Homebrew and (2) pyenv. If you plan to use multiple versions of Python (e.g., 2, 3, and Anaconda), you should use pyenv.

## Sass

While there are a lot of apps out there that get you up and running with Sass (e.g., CodeKit, Prepros), you can also use the command line. To install via Homebrew:

```
brew install sass/sass/sass
```

## Google Cloud

Most of our web apps run on Google Cloud Platform, which lets you build and host applications and websites, store data, and analyze data on Google’s scalable infrastructure.

Visit this site and follow the instructions: [https://cloud.google.com/sdk/docs/quickstart-macos](https://cloud.google.com/sdk/docs/quickstart-macos)
