# Warp

#### **Installation**

My terminal of choice is **Warp** — mainly because it turns the terminal into a modern, structured, and AI-assisted workspace instead of a plain text command line.

To install, simply run:

```zsh
brew install --cask warp
```

After installation, there are three main things we want to set up in a terminal: **Theme**, **Prompt**, and **Shell**.

## **Theme**

#### **Snazzy**

Hyper Snazzy is a nice dark terminal theme originally made for the [Hyper](chatgpt://generic-entity?number=0) terminal. Luckily, they made a Warp version for it [here](https://github.com/GrimLink/warp-theme-snazzy).

Download the [snazzy.yaml](https://github.com/GrimLink/warp-theme-snazzy/blob/main/snazzy.yaml) file and copy it to `~/.warp/themes`

Open Warp, then run the command palette by typing `cmd+p` and search for **Open Theme Picker**. Scroll down and select **Snazzy.**
